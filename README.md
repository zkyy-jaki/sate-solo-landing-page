# 🍢 Warung Sate Solo

A modern restaurant landing page built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**.

This project is developed as a real-world portfolio website for a local restaurant business. It focuses on clean UI, responsive design, SEO, and performance.

---

## ✨ Features

- Modern Landing Page
- Responsive Design
- Mobile Navigation
- Hero Section
- Featured Menu
- About Section
- Gallery
- Testimonials
- FAQ
- Contact Section
- Google Maps Integration
- WhatsApp CTA
- SEO Optimized
- Fast Loading
- Accessibility Friendly

---

## 🛠 Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## 🚀 Getting Started

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
